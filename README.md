# Proiect-Pulverizator-
Pulverizator de apa pentru vara, amplasat in parcuri cand este canicula. Miniatura

Se foloseste:
- 1 x Arduino
- 1 x Releu
- 1 x Water pump(6v)
- 1 x LED
- 1 x DHT11
- 2 x Butoane + Rezistente 10k ohm pentru fiecare
- 1 x LCD cu I2C
- 1 x Baterie externa
